Online Payment Fraud Detection Application

Welcome to Fraud Detection Application!

Fraud prevention in online transactions is critical. This web application helps users detect fraudulent activities in online payment transactions using advanced machine learning models. With an intuitive interface, users can input transaction details and quickly receive predictions about the likelihood of fraud. Whether you're an individual user or a financial professional, this tool provides an easy and effective way to assess transaction risks.

About the Application

The Online Payment Fraud Detection Application leverages cutting-edge machine learning techniques, including Principal Component Analysis (PCA) and XGBoost, to offer accurate fraud predictions. By reducing the number of input features through dimensionality reduction, the application provides a streamlined user experience without compromising accuracy.

Key Highlights:

  • Streamlined User Experience
  • Real-Time Predictions
  • Comprehensive Data Handling

Model Building

Our machine learning models are built using the IEEE-CIS Fraud Detection dataset, ensuring a robust foundation for detecting fraudulent transactions. On the model-building page, you'll find detailed insights into how the models were trained, the features used, and how they contribute to the predictions.

Highlights:

  • Dimensionality Reduction
  • XGBoost Model
  • Clustering for Behavior Analysis

How to Use the Application

Ready to detect fraud? Simply enter the following details:

  1. Transaction type
  2. Amount of the transaction
  3. Old balance and new balance

Click the "Start Application" button, and the model will provide a prediction of whether the transaction is fraudulent.
